# SUPERMARKET_SALES_DASHBOARD

Supemarket Sales Excel Dashboard With Pivot Table and Slicers

**Tools:** MS EXCEL
**Data:** 1000 Sales Record(2019)
**Author:** Rabin Mistri

##Features
5 Pivot Tables
4 Charts(BAR,COLUMN,LINE,PIE)
3Interacive Slices
KPI: Revenue, Orders, Top Branch

## Key Findings
Total Revenue: $322,966
Top Branch: GIZA
Top Category: Food & Beverages
Most Used Payment: Ewallet
